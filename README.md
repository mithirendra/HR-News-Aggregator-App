# HR News Hub
A Python/Flask news aggregator that pulls the latest HR, HR Tech & AI, and Data Science & AI news from multiple RSS feeds and displays them in a clean, categorized web interface. This is part of the HR Digital Transformation Toolkit.

## Features
- Aggregates news from 13+ RSS sources across 3 categories
- Auto-refreshes every hour via APScheduler
- Manual refresh and clear database buttons
- Category filtering tabs
- Clean card-based UI with color-coded source badges
- SQLite database with duplicate prevention

## Categories
- HR News (HR Dive, HR Executive, HR in Asia, HR Asia, The Edge Markets)
- HR Tech & AI News (Unleash, Josh Bersin, e27, Tech in Asia, Digital News Asia)
- Data Science & AI News (Towards Data Science, The Batch, Analytics Vidhya, AI Malaysia)

## Tech Stack
- Python 3
- Flask
- SQLite
- feedparser
- APScheduler
- Jinja2 templates
